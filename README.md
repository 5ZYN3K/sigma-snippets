# SigmaSnippets

A collection of useful VS Code snippets for:

- C#
- .NET MAUI
- JavaScript / TypeScript
- React
- Node.js
- Bootstrap
- XML / XAML

## Features

- Fast MAUI snippets
- CRUD templates (Node + Express + MySQL)
- UI components boilerplate (Bootstrap, React)
- Data structures (Stack, Queue, HashSet, LinkedList)
- API sheets
- Productivity shortcuts

## Usage

Type prefix like:

- ^MAUIlistview
- ^MAUIbutton
- ^MAUIlabel
- ^nodecrud
- ^mysqlcrud
- ^bs-container

and press TAB.
