# SigmaSnippets

A collection of useful VS Code snippets for:

- C#
- .NET MAUI
- JavaScript / TypeScript
- React
- XML / XAML

## Features

- Fast MAUI snippets
- CRUD templates
- UI components boilerplate
- Productivity shortcuts

## Usage

Type prefix like:

- ^MAUIlistview
- ^MAUIbutton
- ^MAUIlabel

and press TAB.
