# SigmaSnippets

A collection of useful VS Code snippets for:

- C#
- .NET MAUI
- JavaScript / TypeScript
- React
- Node.js + Express
- MySQL / mysql2
- Bootstrap
- XML / XAML

## Features

- MAUI UI snippets
- CRUD templates
- Node.js + MySQL API templates
- Data structures
- Bootstrap layouts
- Productivity shortcuts

## Usage

Type a prefix and press `TAB`.
